【Android方面】里面主要是四大模块：

1,主要提供一些常用控件的知识点详解 

2,常用控件的一些封装，

3,app开发中常用功能的集成

a,二维码的扫描与生成功能 b,圆形头像的功能 

4,著名的一些开源框架的使用以及封装

【JAVA方面】java知识点：

1,设计模式：单例的恶汉式和懒汉式写法。

2,Map集合的存取数据详解。


里面包含工具类：

1,正则提供验证邮箱、手机号、电话号码、身份证号码、数字等方法

2,log打印日志的工具类

3，Toast封装工具类

4，Activity的管理工具类

5，通过url获取Json数据 // 通过url获取网络Bitmap数据 （httpClient所完成）

6,网络状态的判断（是否有网络，以及网络是3G,WIFI等状况）
